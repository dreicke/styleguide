Instructions for Photoshop Export

  1. For 75x75 (large) icons, open icons_lg.psd. Or for 45x45 (medium) icons, open icons_md.psd.
  2. Make sure Photoshop's "Layers" window is open. If not, go to Windows on the menu bar and check "Layers"
  3. From this window, find a folder called "icons" and select the icon you'd like to use by clicking the empty box on the left of its name. This will turn on its visibility, indicated by an eye.
  4. Next, select the background color you'd like to use from "backgrounds" folder in the same way.
  5. In the display area, you should see a rendering of the icon you'd like to use.
  6. Export this icon by selecting "Save for Web" from the File menu.
  7. You will have a few options here. Choose PNG-8 from the first menu. Select "No Dither". Check "Transparency". Then select "No Transparency Dither".
  8. The preview you see now should be crisp-looking and about twice as big as you want it to appear on the web. If it looks correct, click Save, and choose the directory where you'd like to pull icons from.

How to use these icons in code:

While slightly more costly for load time, we found that, on Retina displays, there is a significant difference in clarity if we use icon images 2x the size of display and then shrink them with CSS. So when using these, please specify a width.

<img style="width:45px" src="[folder]/[filename].png" />
